background-image-repeat has three value. By default, the background-image property repeats an image both horizontally and vertically.
    For repeating horizontally only use (background-repeat: repeat-x;)
    For repeating vertically only use (background-repeat: repeat-y;) 
    Showing the background only once is specified by the (background-repeat: no-repeat)

The background-position property is used to specify the position of the background image.
